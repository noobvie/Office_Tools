ℹ How it works
Select any files — they are bundled into a .zip archive using DEFLATE compression entirely in your browser via JSZip. The archive is generated in memory and downloaded directly — no upload, no server, no account.

Tip: For files already compressed (JPG, MP4, MP3, PDF) uncheck Compress (DEFLATE) — it won't reduce their size and wastes time.

ℹ Supported formats
This tool extracts standard ZIP archives (including those created by Windows, macOS, Linux, 7-Zip, WinRAR, and this tool). Password-protected ZIP files are not supported. For other formats (.rar, .7z, .tar.gz), use a desktop application like 7-Zip (free).
