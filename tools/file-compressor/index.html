About File Compressor & Extractor

Need to shrink the size of an image or document before sharing or uploading? This File Compressor handles images (JPEG, PNG, WebP) and PDFs right in your browser. The key difference from most online compression tools: nothing is uploaded to a server. All processing happens locally, so your files stay completely private.

Drag in your file, adjust the quality or compression level, and download the result. Perfect for reducing email attachment sizes, optimizing images for web pages, and shrinking PDFs before uploading to a portal or cloud storage. No account, no watermarks, no surprise size limits.

ℹ How it works
Select any files — they are bundled into a .zip archive using DEFLATE compression entirely in your browser. Optionally set a password for AES-256 encryption — compatible with 7-Zip, WinRAR, and Windows. The archive is generated in memory and downloaded directly — no upload, no server, no account.

Tip: For files already compressed (JPG, MP4, MP3, PDF) uncheck Compress (DEFLATE) — it won't reduce their size and wastes time.

ℹ Supported formats
This tool extracts standard ZIP archives (including those created by Windows, macOS, Linux, 7-Zip, WinRAR, and this tool). Password-protected ZIP files using AES-256 and ZipCrypto encryption are fully supported — just drop the file and enter the password when prompted. The password never leaves your browser. For other formats (.rar, .7z, .tar.gz), use a desktop application like 7-Zip (free).
