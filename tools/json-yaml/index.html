Convert between JSON and YAML in either direction, with live validation. Type on either side.

About this tool

Paste JSON on the left or YAML on the right and the other side updates automatically. Toggle the direction with the swap button, then copy the result. Invalid input is flagged with a clear parse error so you can fix it quickly.

Parsing uses the js-yaml library loaded from a CDN. All conversion happens in your browser — your data never leaves the page.
