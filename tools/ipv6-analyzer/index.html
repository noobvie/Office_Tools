About IPv6 Analyzer

IPv6 addresses can be written in multiple equivalent ways — the full expanded form (8 groups of 4 hex digits) and the compressed form (leading zeros and consecutive zero groups replaced with ::). This tool converts between both representations and identifies what type of address you have based on RFC 4291 prefix ranges: Global Unicast (publicly routable), Link-local (fe80::/10, not routable), Unique Local (fc00::/7, private), Multicast (ff00::/8), Loopback (::1), Teredo, 6to4, and more. Also shows the /64 network prefix — the standard allocation unit for IPv6 subnets. Fully local — nothing is sent to a server.
