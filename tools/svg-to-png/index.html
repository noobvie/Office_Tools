About the SVG to PNG Converter

SVG is perfect for the web, but plenty of apps still want a raster image. Drop an .svg file or paste the markup, set a scale (or exact dimensions) and a background (transparent for PNG/WebP, or a solid color), then download a crisp PNG, JPG or WebP. Because SVG is vector, you can export at 2×, 4× or higher with no quality loss. Conversion runs locally — nothing is uploaded.

Note: external resources referenced by the SVG (remote images, web fonts) may not render — embed them in the SVG for a faithful result.
