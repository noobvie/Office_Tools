🛠️ Office Tools YouTube Downloader

📥 YouTube to MP3 / MP4

Download videos as MP4 or audio as MP3 — single videos and full playlists with batch download.

📋 Quick Setup Notes
  • ⚡ Local (default) — uses the office-tools-yt service on this server (yt-dlp + ffmpeg, installed via deploy.sh step 1, auto-updated weekly). Uses Android/TV player clients to bypass bot-detection — no cookies needed in most cases. MP4 up to 1080p + true MP3.
  • Cobalt — cobalt API. api.cobalt.tools now requires an API key — get one free at cobalt.tools and set it in Advanced, or self-host cobalt without auth.
  • Invidious — independent fallback, no yt-dlp needed. Video up to ~720p, audio as M4A/WEBM (not MP3).
  • Playlist — requires a free YouTube Data API v3 key (shown automatically when a playlist URL is pasted).


Local (yt-dlp) — your self-hosted yt-dlp server. MP4 up to 1080p + true MP3. Fastest — no external dependency.

Stored in your browser only. Get a free key at Google Cloud Console → enable "YouTube Data API v3" → Credentials → Create API Key. Free quota: 10,000 units/day (a full playlist load uses ~1–2 units).

⚙️ Advanced — API endpoints

Default: /yt-api/ — your self-hosted office-tools-yt service (Node.js + yt-dlp + ffmpeg). Deployed automatically by deploy.sh step 1. Supports MP4 up to 1080p and true MP3 audio. No external API, no rate limit, no third-party dependency.

api.cobalt.tools now requires auth. Either: (a) get an API key from cobalt.tools (free) and paste it above, or (b) self-host cobalt without auth and set your instance URL above, or (c) use Local backend (recommended — no external dependency).

Default: https://inv.tux.pizza — any public Invidious instance works. Self-host for reliability. Video up to ~720p, audio as M4A/WEBM (not MP3). Useful as independent fallback when Local is unavailable.
