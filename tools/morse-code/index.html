📡 Morse Code Translator

Translate text to Morse code and back. Play it as audio beeps with a visual flash.

About this tool

Convert text to Morse code and decode Morse back to text using International Morse. Dots and dashes are separated by spaces, and words by a forward slash (/). Hit Play to hear it as audio beeps and watch the lamp flash in time — adjustable from slow to fast (words per minute).

Supports letters, numbers and common punctuation. Audio is generated with the Web Audio API; everything runs locally.
