About Email Validator

This tool runs the checks a mail server does before it ever tries to deliver a message: it confirms the address is syntactically valid, that its domain publishes MX records (so it can actually receive email), and whether the domain is a known disposable / throwaway provider or the local part is a role-based mailbox (like info@ or support@) rather than a real person.

It does not send a probe email or do SMTP RCPT TO verification, so it cannot tell whether one specific mailbox exists — that would require connecting to the recipient's mail server, which most providers reject or rate-limit. For a full picture of a domain's email setup (SPF, DKIM, DMARC), use the Email Deliverability Checker. To see if a mail server's IP is on a spam blocklist, use the DNSBL Blacklist Checker.
