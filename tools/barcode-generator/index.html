About this tool

Create scannable 1D barcodes for products, inventory, and labels. Choose a format — Code 128, EAN-13, EAN-8, UPC-A, Code 39 or ITF-14 — type your data, and the barcode renders instantly. Adjust bar width, height, colors and the human-readable text, then download as PNG or SVG.

Rendering uses the JsBarcode library loaded from a CDN. Everything runs in your browser — nothing is uploaded. Need a 2D QR code instead? Use the QR & Barcode Generator.
