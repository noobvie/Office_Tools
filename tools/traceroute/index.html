About Online Traceroute

Traceroute (also called tracert on Windows) maps every router hop between our server and your target host by sending packets with increasing TTL values. Each line of output shows a hop number, the router's IP or hostname, and round-trip times — letting you pinpoint where latency spikes or where packets stop. Full IPv4 and IPv6 support: select IPv6 mode to force a traceroute over IPv6 (equivalent to traceroute6 / tracert -6). Results stream live as each hop responds, so you don't have to wait for the full trace to complete. Because the trace runs from our server, the path reflects server-to-target routing, not your local network path.
