About this tool

A two-way code formatter for front-end developers. Beautify compressed or inconsistent HTML, CSS, and JavaScript into readable, properly-indented code, or minify it to strip whitespace and comments and shrink the payload. Pick the language, pick the direction, and the result updates with a byte-count comparison.

Beautification is powered by the well-known js-beautify library loaded from a CDN; everything runs in your browser, so your code is never sent to a server.
