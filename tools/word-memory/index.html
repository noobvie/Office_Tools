About Word Memory

Word Memory tests your recognition memory — you're shown a list of words briefly, then asked to identify which ones you saw from a larger mixed list that includes new distractor words. It's a classic cognitive test of verbal recall and recognition accuracy.

A useful quick cognitive warm-up or active mental break during the workday. Recognizing previously seen items from a distractor set engages a different part of your memory than recall-from-scratch tasks. See how many words you can reliably identify — and whether your score improves with a little practice.
