Decode color bands into a resistance value, or turn a value back into bands. Supports 4, 5 and 6-band resistors.

About resistor color codes

Through-hole resistors print their value as colored bands. The first bands are significant digits, the next is a multiplier, then a tolerance band, and on 6-band parts a final temperature coefficient. A 4-band resistor has 2 digit bands; a 5/6-band has 3. This tool reads in both directions — pick the band colors to get the resistance, or type a resistance to see the band colors.

Read the resistor with the tolerance band (often gold or silver) on the right. Gold/silver are never used for the first band, which removes the ambiguity about which end to start from.
