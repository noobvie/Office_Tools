About this tool

Translate text into 8-bit binary and decode binary back into readable text. Every character is encoded as UTF-8 bytes, so emoji and accented characters work correctly. The converter also shows the hexadecimal and decimal byte values for the same input.

Type in either box — the rest update live. Everything runs locally in your browser.
