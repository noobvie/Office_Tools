About the Website Status Checker

Enter any website address and this tool fetches it from our server to tell you if it is up or down. Because the request comes from our network rather than yours, it answers the classic question — "is this site down for everyone, or just me?". If we report the site as up but you still cannot load it, the problem is on your side: your ISP, DNS resolver, VPN, or a local firewall.

The result shows the HTTP status code (2xx success, 3xx redirect, 4xx client error, 5xx server error), the response time in milliseconds measured from our server, the final URL after any redirects, the reported Server header, and — for HTTPS sites — the SSL certificate issuer and days until expiry so you can catch certificates about to lapse. To inspect the network path itself, try Ping or Traceroute; to view a page that is blocking your region, use the Web Proxy.
