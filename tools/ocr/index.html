About Image to Text (OCR)

Optical Character Recognition (OCR) reads the text inside an image and turns it back into editable, copyable text. Drop in a photo, screenshot, or scanned page, pick the language, and the tool extracts the text for you to copy or download. It works on receipts, book pages, business cards, signs, slides, and PDF screenshots.

Recognition runs locally using the open-source Tesseract engine compiled to WebAssembly — your image is never uploaded to any server. The language data file downloads once from a CDN on first use and is then cached. For best results use a sharp, well-lit, straight image; tiny or skewed text will read less accurately.
