About the Web Proxy

This reader-mode web proxy fetches a web page from our server and shows it to you, so the destination site sees our IP address instead of yours. That makes it handy when a website blocks your IP or geographic region, or when you just want to read a page without connecting to it directly. Links and images on the page are rewritten to keep routing through the proxy as you click around.

Because it works by fetching and rewriting HTML, it suits articles, documentation and mostly static pages. It is not a full browser: JavaScript-heavy single-page apps, logins, video players and sites with strict anti-bot protection (Cloudflare challenges, CAPTCHAs) will not render reliably. Scripts are stripped by default for safety — turn that off only if a page genuinely needs them. This is not anonymity software like Tor; don't send passwords or sensitive data through it. To simply check whether a site is online, use the Website Status checker.
