About DNS Lookup

This tool lets you query any DNS record type for any domain — the equivalent of running nslookup or dig from your terminal, but in your browser. Results are fetched via Cloudflare DNS-over-HTTPS (1.1.1.1) for fast, accurate, privacy-respecting queries. Use the AAAA tab to check if a domain has IPv6 addresses configured. Use MX to inspect mail server routing, TXT for SPF / DKIM / DMARC records, and CAA to see which certificate authorities are authorized to issue SSL for the domain. Switch between record types instantly without re-entering the domain name.
