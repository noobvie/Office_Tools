🔒 Privacy & Cookie Notice

Plain-English summary of what happens to your data when you use Office Tools.

Last updated: 1 June 2026

The short version: The content you type or drop into a tool (text, images, files) is processed in your browser and is not uploaded — with the few clearly-marked exceptions listed below (URL Shortener, Pastebin, File Share, the Network tools, and the YouTube Downloader), which must send your input to a server to work. We use Google Analytics to count visits, which sets cookies. We have no accounts, no logins, and we never sell data.

1. What stays in your browser

The large majority of tools run 100% client-side. Your input never leaves your device. This includes (among others) the password generator, hash generator, JSON/CSV/Base64/encoding tools, PDF Toolkit, Image Converter, Image Cropper, Image-to-PDF, OCR, calculators, and all games. For these tools we receive nothing about your content.

Some tools save small amounts of data in your browser's localStorage purely so the tool works (e.g. your theme choice, Notepad notes, prompt templates). This data stays on your device and is never transmitted to us.

2. What is sent to our server (and why)

A small number of tools cannot work without sending your input to a backend. When you use these, you are knowingly submitting that data:

ToolWhat is sentRetention
URL ShortenerThe URL you shorten + optional aliasUntil expiry you set
PastebinThe text/code you pasteUntil expiry / burn-after-read
File ShareThe file you uploadAuto-deleted (≈7 days)
Port Checker, Ping, TracerouteThe host/IP you enterNot stored — run and discarded
Domain CheckerThe domain you query (WHOIS/RDAP/DNS)Not stored
DNS & Reverse DNS LookupThe domain/IP — resolved via Cloudflare DNS-over-HTTPSNot stored by us
YouTube DownloaderThe video/playlist URL — sent to our self-hosted cobalt serverNot stored
Feedback formYour message + the page you sent it fromStored so we can read it

We also record an anonymous per-tool view count (the tool name only — no personal data) to know which tools are popular.

3. Cookies & analytics

We use Google Analytics 4 to understand how many people visit and which tools are used. To do this, Google sets cookies in your browser:

CookieSet byPurposeTypical lifetime
_gaGoogle AnalyticsDistinguishes unique visitors2 years
_ga_<ID>Google AnalyticsMaintains session state2 years

Google Analytics may process your approximate location (derived from your IP address), device, browser, and the pages you view. It does not receive the content you process inside the tools. See Google's Privacy Policy.

We do not use advertising cookies, retargeting pixels, or sell any data.

4. Third-party services

To keep the site fast and free, some tools load code or data from third parties. These providers may see your IP address as part of serving the request, but not your tool content:

  • Google Analytics — visit statistics (cookies, as above)
  • CDNs (jsDelivr, unpkg, Cloudflare) — deliver libraries used by tools such as OCR, PDF, and image tools
  • Cloudflare DNS-over-HTTPS — powers the DNS Lookup tool
  • Public price / IP / geolocation APIs — power the Currency, What-Is-My-IP, and IP Location tools

5. How to opt out

  • Install Google's Analytics Opt-out Browser Add-on.
  • Block third-party cookies, or use a privacy extension / ad blocker — the tools still work.
  • Use your browser's "clear cookies and site data" to remove the analytics cookies at any time.

6. Children

Office Tools is a general-purpose utility site and is not directed at children under 13. We do not knowingly collect personal information from children.

7. Changes

If we change what we collect, we will update this page and the "last updated" date above.
